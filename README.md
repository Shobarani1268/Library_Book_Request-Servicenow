# Library Book Request - ServiceNow Project

## Project Overview
A ServiceNow application to manage library book requests.

## Features
- Auto-generated request number (LBR)
- Student can submit book requests
- Date validation (no past dates)
- Email notification to librarian

## Components Created
- Custom Table: u_library_book_request
- Business Rule: Validate Issue Date
- Email Notification: New Submission Alert

## How to Import
1. Download the XML files
2. In ServiceNow go to System Update Sets
3. Import XML file
4. Preview and Commit
